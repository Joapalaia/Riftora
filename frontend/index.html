Estatísticas do seu Invocador Your Summoner Statistics
⚡
Pode demorar um pouco
O servidor dorme quando fica inativo. A primeira requisição pode levar até 60 segundos para acordar. Nas buscas seguintes, tudo vai muito mais rápido.
🔑
API Key de Desenvolvimento
Usamos a chave básica da Riot API, que tem limite de requisições. Buscar muitas partidas pode levar alguns minutos — quanto mais partidas, mais tempo.
💾
Dados em Cache
Após a primeira busca, os dados ficam salvos. Próximas consultas ao mesmo jogador trazem só as partidas novas, muito mais rápido.
⚡
May take a moment
The server sleeps when idle. The first request can take up to 60 seconds to wake up. Subsequent searches will be much faster.
🔑
Development API Key
We use Riot's basic API key, which has request limits. Fetching many matches can take a few minutes — the more matches, the longer it takes.
💾
Cached Data
After the first search, data is saved. Future lookups for the same player only fetch new matches — much faster.
